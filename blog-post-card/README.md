# Blog Post Card

## Lab Project
A simple blog post card built with HTML and CSS.

## Skills Practiced
- HTML structure
  First time writing up HTML with just general guidelines on what I should do rather than following implicit instructions. 
- CSS selectors
- Images
- Border radius
- Spacing (margin & padding)
  Unfortunately I still mix up the two so practise practise..

## Challenges
- Scaling an image to the size of a specific containing element.
- Learning the difference between `border` and `outline` and the limitations/advantages. 

## What I Learned
- How to make an image fill its container.
- How pseudo-classes like `:hover` and `:focus` work.
  Especially in terms of the order necessary to fully display the effect.
- How to style links to look like buttons.
